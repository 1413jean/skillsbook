# SkillsBook · Skills 與 MCP 功能工具書

> Know your tools, and the tools will work for you.

一本給 Claude Code 使用者的「功能工具書」——把所有 **Skills** 與 **MCP 連接服務** 整理成可以翻閱的書：照「想做什麼」找工具，不用記名稱。內建 200+ 條目，每一條都有**英文標題＋中文介紹**、觸發方式、移除方式。

**單一 HTML 檔案**，沒有 build、沒有依賴（動畫用 CDN 的 GSAP）。下載就能開，丟上 GitHub Pages 就能分享。

---

## Quick Start

**Mac（最順）** — 雙擊 `start.command`
> 自動開伺服器＋瀏覽器，同步檢查會「自動掃描」你的工具。
> 第一次被 macOS 擋下：對它**按右鍵 → 打開**一次就好。

**任何系統（免安裝）** — 直接雙擊 `index.html`
> 用瀏覽器打開就能看，同步改走「選資料夾」流程（需要 Chrome／Edge）。

**想放網路上分享** — 推上 GitHub 開 Pages，別人開網址即用。

> ⚠️ 動畫用線上的 GSAP，需要網路；斷網頁面還能看，只是少了動畫。

---

## 怎麼查看自己安裝的 Skills 與 MCP

到目錄最下方「**附錄 · 工具**」→ 點「**同步檢查**」，它會找出你裝了、但工具書還沒收錄的項目。視窗會自動判斷用哪種方式：

**A. 有開伺服器（雙擊 `start.command` 的人）**
打開同步視窗 →  **自動掃描**，什麼都不用做。

**B. 沒開伺服器（直開 HTML 的人，需 Chrome／Edge）**
1. 點「**📂 選取 .claude 資料夾**」
2. 照視窗指示前往 `~/.claude`
   （Mac 按 `Cmd+Shift+G` 貼路徑；Windows 路徑列貼 `%USERPROFILE%\.claude`——路徑點一下就複製）
3. 允許瀏覽器讀取 → 自動掃出清單

掃到之後：
- 點「**＋ 加入**」或「**全部加入**」收進「**新收錄**」章節
- 進該章節 → 「**複製翻譯 Prompt**」貼給 Claude → 回傳貼回「**貼上翻譯結果**」，中文介紹就補上了
- 每條都能「**編輯中文介紹**」微調

> 資料夾選取讀不到 symlink 形式的 skill（瀏覽器安全限制），這類項目請改用伺服器方式（`start.command` 或 `php -S 127.0.0.1:4173`）。

## 功能

- **目錄翻書式導覽**——上篇 Skills、下篇 MCP，點章節進入、左右滑動翻頁
- **搜尋**——按 `/` 聚焦，中英文與觸發語都搜得到
- **深淺色模式**——自動記住偏好
- **同步檢查**——掃描你安裝的 skills 與 plugins，找出未收錄的項目（伺服器自動掃描／免伺服器資料夾選取）
- **一鍵匯入**——把掃到的新工具收進「新收錄」章節（存在瀏覽器 localStorage）
- **翻譯工作流**——只有英文介紹時，複製 prompt 給 Claude，把回傳 JSON 貼回即套用中文
- **可編輯**——匯入項目的中文名稱與介紹可直接在頁面上編輯
- **移除教學**——每個條目附一段可複製的 prompt，貼給 Claude 就能請它幫你移除該工具
- **匯出／匯入**——個人資料（匯入項目＋隱藏設定）打包成 JSON 帶走或分享
- **隱藏／復原**——不需要的條目可以隱藏，附錄一鍵全部復原
- **附錄工具列**——同步、匯出、匯入、復原整合在目錄最下方的「附錄 · 工具」

## 結構

```
skillsbook/
├── index.html      # 整個網站（CSS + 資料 + JS 全部內嵌）
├── start.command   # Mac 雙擊啟動本機伺服器＋開瀏覽器
├── sync.php        # 本機同步端點（選用，伺服器自動掃描用）
└── README.md
```

條目資料在 `index.html` 的 `DATA` 陣列，章節定義在 `PARTS`。要長期維護就直接編輯這兩處。
