# Claude Toolbook · 功能工具書

> Know your tools, and the tools will work for you.

一本給 Claude Code 使用者的「功能工具書」——把所有 **Skills** 與 **MCP 連接服務** 整理成可以翻閱的書：照「想做什麼」找工具，不用記名稱。內建 200+ 條目，每一條都有**英文標題＋中文介紹**、觸發方式、移除方式。

**單一 HTML 檔案**，沒有 build、沒有依賴（動畫用 CDN 的 GSAP）。下載就能開，丟上 GitHub Pages 就能分享。

## 使用方式

```bash
# 方法一：直接開
open index.html

# 方法二：本機伺服器（同步檢查的 PHP 來源需要它）
php -S 127.0.0.1:4173
```

## 功能

- **目錄翻書式導覽**——上篇 Skills（16 章）＋下篇 MCP（5 章），點章節進入、左右滑動翻頁
- **搜尋**——按 `/` 聚焦,中英文與觸發語都搜得到
- **深淺色模式**——自動記住偏好
- **同步檢查**——掃描你自己安裝的 skills 與 plugins，找出工具書還沒收錄的項目：
  - 來源 A：`sync.php`（需本機 PHP 伺服器）
  - 來源 B：**免伺服器**——瀏覽器資料夾選取（Chrome／Edge），直接選 `~/.claude` 或 `~/.claude/skills`
- **一鍵匯入**——把掃到的新工具收進「新收錄」章節（存在瀏覽器 localStorage）
- **翻譯工作流**——匯入的項目只有英文介紹時：點「複製翻譯 Prompt」→ 貼給 Claude → 把回傳的 JSON 貼回「貼上翻譯結果」即套用中文
- **可編輯**——匯入項目的中文名稱與介紹可直接在頁面上編輯
- **移除教學**——每個條目附一段可複製的 prompt，貼給 Claude 就能請它幫你移除該工具
- **匯出／匯入**——個人資料（匯入項目＋隱藏設定）打包成 JSON 帶走或分享
- **隱藏／復原**——不需要的條目可以隱藏，頁尾一鍵全部復原

## 同步自己的 Skills

1. 捲到頁尾，點「**同步檢查**」
2. 沒有 PHP 伺服器的話，點「**📂 改用資料夾選取**」，選 `~/.claude` 資料夾
   （macOS 的 Finder 對話框按 `Cmd+Shift+G` 輸入 `~/.claude`）
3. 在清單裡點「**＋ 加入**」或「**全部加入**」
4. 進「新收錄」章節 → 點「**複製翻譯 Prompt**」→ 貼給 Claude → 回傳結果貼回「**貼上翻譯結果**」

> 注意：資料夾選取讀不到 symlink 形式的 skill（瀏覽器安全限制），這類項目請改用 PHP 來源。

## 結構

```
claude-toolbook/
├── index.html   # 整個網站（CSS + 資料 + JS 全部內嵌）
├── sync.php     # 本機同步端點（選用）
└── README.md
```

條目資料在 `index.html` 的 `DATA` 陣列，章節定義在 `PARTS`。要長期維護就直接編輯這兩處。
